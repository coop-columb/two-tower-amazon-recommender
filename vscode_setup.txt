VS Code Manual Setup Instructions:

1. OPEN WORKSPACE:
   - Use Finder to navigate to project folder
   - Right-click folder -> "Open with Code"
   OR
   - Open VS Code app directly
   - File -> Open Folder -> Select two-tower-amazon-recommender

2. VERIFY PYTHON INTERPRETER:
   - Check bottom-left corner of VS Code
   - Should show: Python 3.11.7 ('venv': venv)
   - If not: Cmd+Shift+P -> "Python: Select Interpreter"
   - Choose: ./venv/bin/python

3. TERMINAL ACTIVATION:
   - Open VS Code integrated terminal: Ctrl+`
   - If (venv) not shown in prompt: source activate_dev.sh

4. WORKSPACE READY INDICATORS:
   - Bottom status shows correct Python version
   - Terminal shows (venv) prefix
   - File explorer shows project structure
